## Getting Started
1. Generate the token with [Personal access tokens](https://github.com/settings/tokens?type=beta)
2. Set the token in local.properties

> local.properties
```
sdk.dir=/Users/user_name/Library/Android/sdk
GITHUB_API_TOKEN=token
```
